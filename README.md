# 웹프론트엔드 면접 전 과제
- **실행 방법**
  - 터미널에 `yarn run dev` 입력 후 localhost:5173으로 접속
- **기능**
  - 책 정보 추가
  - 추가된 책 리스트 확인하기
- **사용한 기술**
  - React + Typescript + Vite 사용
  - react-router-dom을 사용하여 주소에 맞는 페이지 호출
  - GitHub 칸반 작성 및 git flow 전략
